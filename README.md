# My Portfolio Website

Welcome to the repository for my personal portfolio website! 🚀 This website is created using HTML, CSS, and JavaScript to showcase my professional journey, technical projects, and skills.

## Features

- 🌐 **Responsive Design:** Ensures seamless viewing across various devices.
- 💡 **Project Showcase:** Detailed descriptions and tech stacks of my notable projects.
- 📝 **Blog Section:** Explore my thoughts and experiences.
- 📧 **Contact Form:** Easy way to get in touch.

## Technologies Used

- **HTML:** Structure of the web pages.
- **CSS:** Styling for a visually appealing design.
- **JavaScript:** Adding dynamic and interactive elements.
- **Bootstrap:** Frontend framework for enhanced styling.
