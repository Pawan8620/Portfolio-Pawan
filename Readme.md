### 🚀 **Getting Started**  
To run this project locally:  
1. Clone the repository:  
   ```bash
   git clone https://github.com/Pawan8620/Portfolio-Pawan.git
   ```
2. Navigate to the project directory:  
   ```bash
   cd Portfolio-Pawan
   ```
3. Open `index.html` in your browser:  
   ```bash
   start index.html
   ```
